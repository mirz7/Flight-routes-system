# Django Flight Routes System

Django web application for managing airport routes and flight connections.

## Features

- **Add Routes** - Create airport routes with code, position, and duration
- **Find Nth Node** - Search for the Nth left/right node from an airport
- **Find Longest Route** - Get the maximum duration route from an airport
- **Find Shortest Route** - Get the minimum duration route between two airports


## Tech Stack

- Python 3.x
- Django 4.x
- SQLite (default)
- HTML/CSS
